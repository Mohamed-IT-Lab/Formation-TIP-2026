# Cours 00 : Le Réseau, c'est comme le Code de la Route

Pour comprendre comment les données voyagent d'un PC à un serveur, il suffit de regarder comment une voiture voyage d'une maison à un magasin.

## 1. L'Adresse IP = La Plaque d'immatriculation
Chaque appareil sur le réseau doit avoir une identité unique. 
* Sans plaque, on ne peut pas circuler. 
* Si deux voitures ont la même plaque, c'est le conflit (Conflit d'IP) !

## 2. Le Switch = L'Intersection (Le quartier)
Le **Switch** relie les ordinateurs d'un même bureau. 
* C'est comme les rues d'un même quartier.
* On roule à 30 ou 50 km/h, c'est le réseau local (LAN).

## 3. Le Routeur = L'Échangeur d'Autoroute
Le **Routeur**, c'est la machine qui permet de changer de ville. 
* C'est lui qui possède la **"Table de Routage"** (Le GPS).
* Si vous voulez aller sur Internet (une autre ville), vous devez passer par le Routeur.

## 4. La Passerelle (Gateway) = La bretelle d'accès
C'est la porte de sortie de votre quartier. 
* Si votre PC ne connaît pas l'adresse de la Passerelle, il reste bloqué dans son quartier (son réseau local). 
* Il peut parler à ses voisins, mais il ne peut pas aller sur Google.

## 5. Les Paquets = Les Camions de livraison
Sur Internet, on n'envoie pas un fichier d'un seul coup. On le découpe en petits **paquets**.
* Chaque paquet a une adresse de départ et une adresse d'arrivée.
* Ils peuvent prendre des routes différentes, mais ils arrivent tous au même endroit pour être remontés.

## Résumé du code de la route informatique :
| Élément Informatique | Équivalent Routier |
| :--- | :--- |
| **Carte Réseau** | Le moteur de la voiture |
| **Câble Ethernet** | La route bitumée |
| **Adresse IP** | L'adresse postale / La plaque |
| **Routeur** | Le panneau de direction / GPS |
| **Ping** | "Klaxonner" pour voir si quelqu'un répond |
