# Terminal, Invite de commande et Processus informatique

En informatique, on passe souvent de l'interface graphique (GUI) aux lignes de commande. Voici comment s'y retrouver et comprendre comment la machine réagit.


## I. Les outils de ligne de commande

### 1. L'Invite de commande (Command Prompt / CMD)
Sous Windows, le programme classique (`cmd.exe`) s'appelle l'**Invite de commande**.
* **Origine :** Son nom vient de l'époque de MS-DOS.
* **Pourquoi "Invite" ?** On l'appelle ainsi car le curseur clignotant "invite" l'utilisateur à taper une instruction (on parle aussi de "Prompt").

### 2. Le Terminal
À l'origine, le terme **Terminal** désigne un appareil physique (écran + clavier) relié à un gros ordinateur central (Mainframe).
* **Usage moderne :** Dans l'univers Linux et macOS (basés sur Unix), on utilise "Terminal" pour désigner l'application qui permet de taper des commandes. C'est devenu le terme universel dans le monde du logiciel libre.

### 3. La confusion moderne : Windows Terminal
Microsoft a sorti récemment une application appelée **Windows Terminal**. 
* C'est une application "conteneur" qui peut ouvrir plusieurs onglets en même temps : l'Invite de commande (CMD), PowerShell, et même du Linux via **WSL**.
* ** **WSL** signifie *Windows Subsystem for Linux* (Sous-système Windows pour Linux).



## II. Les étapes du processus informatique

Au niveau le plus élémentaire, tous les ordinateurs fonctionnent selontrois étapes :


| **1. L'Entrée** | **Input** | Tout commence par une action de l'utilisateur (clic souris, touche clavier, écran tactile). |
| **2. Le Traitement** | **Processing** | Les composants internes prennent le relais. L'OS traduit votre demande et ordonne au matériel de l'exécuter. |
| **3. La Sortie** | **Output** | C'est le résultat concret (affichage à l'écran, impression d'un document, son dans les enceintes). |



Si vous tapez une lettre au clavier (**Input**), que rien ne s'affiche (**Output**) mais que le PC chauffe (**Processing**), c'est probablement que le système d'exploitation est "planté" ou occupé.
